##### 🏩 "savePath": null, --> null이면 scss파일과 같은 위치에 style.css가 생긴다.
##### ⛺ "savePath": "~/css", --> ~ 은 style.scss를 의미, /는 style.scss 있는 폴더를 의미, scss 파일과 같은 위치에 css 폴더가 생성되고 그 css폴더 안에 컴파일된 style.css가 생긴다.
